# Lunar Drago 404

# Usage
The following assumes the use of `node@>=18` and `foundry`.

## Setup
Create a .env file following .env.example pattern:
  ```
  cp .env.example .env
  ```

## Install Dependencies
Install node dependencies:
```
yarn install
```
Install foundry libs:
```
forge install
```

## Compile Contracts

```
forge compile
```

## Test Contracts

```
forge test
```
